
life-management，顾名思义，是用于自我管理的。包括但不限于睡眠管理、任务管理、时间管理、读书、运动等功能，可根据自己需要灵活扩展。

本系统采用shell+html+echars构建，具备了db，展示页面和统计报表，在命令行操作，页面即时更新，可以随时在浏览器浏览每日数据，当月数据，当年数据等6个页面。
